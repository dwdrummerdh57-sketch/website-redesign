Long’s Plumbing Website Redesign

A modern, responsive landing page redesign for a local plumbing service company.
This project focuses on clean UI/UX, strong visual hierarchy, and real-world layout structure using HTML and CSS.



https://dwdrummerdh-sketch57.github.io/website-redesign/

Project Overview

This project was built as a portfolio piece to demonstrate front-end fundamentals and real-world layout design.
The goal was to take a typical outdated service website and redesign it into a modern, conversion-focused landing page.

Key design goals:

Improve readability and visual hierarchy
Create strong call-to-action sections
Build a clean, professional layout structure
Ensure responsiveness across devices
Features
Responsive layout (desktop, tablet, mobile)
Modern hero section with clear value proposition
Service card grid with improved content structure
“Why Choose Us” section with supporting visuals
Conversion-focused CTA sections
Clean footer with structured navigation

Technologies Used
HTML5 (semantic structure)
CSS3 (Flexbox, Grid, responsive design)
VS Code
Git & GitHub

What I Learned
Structuring real-world layouts using semantic HTML
Managing spacing, alignment, and visual hierarchy
Building responsive designs using media queries
Debugging layout issues using browser DevTools
Organizing CSS for scalability and readability
Challenges & Solutions

Challenge: Maintaining consistent spacing across sections
Solution: Implemented container-based layout and spacing system

Challenge: Responsive behavior across breakpoints
Solution: Used targeted media queries at 1024px, 768px, and 480px

Challenge: Balancing typography and layout proportions
Solution: Iteratively refined font sizes, max-widths, and spacing
